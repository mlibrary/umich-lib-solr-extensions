# Example Schema Fragments

Sample Solr `schema.xml` fragments demonstrating `umich-lib-solr-extensions` field types and filters.

- `authority_browse.xml` -- `AnalyzedString`-based browse schema for authority records
- `callnumber_browse.xml` -- Call number field types and starts-with analysis chain

These are documentation examples only; they are not deployed by the integration tests.
See [../../src/test/resources/solr-integration/test-core/conf/schema.xml](../../src/test/resources/solr-integration/test-core/conf/schema.xml) for the live test configset.
